# Bioscratch Landing Page

Landing page for [Bioscratch](https://github.com/Broccolito/bioscratch), a what-you-see-is-what-you-get Markdown editor for researchers, data scientists, and AI engineers.

## About Bioscratch

Bioscratch is a desktop Markdown editor built for humans and AI agents alike. Files stay plain Markdown, readable by anyone or anything.

- **GitHub:** [Broccolito/bioscratch](https://github.com/Broccolito/bioscratch)
