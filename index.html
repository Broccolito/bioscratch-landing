The Markdown editor for the age of AI.

A what-you-see-is-what-you-get Markdown editor for researchers, data scientists, and AI engineers. Built for humans and agents alike.

01
Agent & Human Ready
Files stay plain Markdown, readable by any AI model or any person, as-is.
02
What You See Is What You Get
No split panes. No preview mode. Headings, tables, and math render as you type. Click to edit.
03
Minimal by Design
A distraction-free canvas. Nothing between you and the document.

Features

Everything you need. Nothing you don't.

True WYSIWYG
Headings, bold, tables, and task lists render live as you type. Click anywhere to edit raw syntax.
Math & Diagrams
KaTeX inline and block math. Mermaid flowcharts and sequence diagrams, rendered live in the editor.
Code Blocks
Syntax highlighting for Python, R, TypeScript, Julia, Go, Rust, and more.
30+ Themes
GitHub, Material, IBM Carbon, NASA, Spotify, and more. Customizable YAML-based theme files.
Multi-Tab & Multi-Window
Drag tabs to detach into separate windows. Autosave every 30 s. Crash recovery on next open.
Export
HTML (self-contained, KaTeX + Mermaid bundled) and PDF via Pandoc with proper typesetting.
File Format Support
Opens .md, .txt, .py, .r, .json, .yaml, .toml, .tex, .sql, .html, .ts, .go, .rs, and more.
Mermaid Diagrams
Visualize workflows, pipelines, and skill maps with live Mermaid diagrams. Great for agents and researchers.
